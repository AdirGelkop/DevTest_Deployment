# Linux Cheatsheet

## תהליכים (Processes)

| פקודה | מה עושה |
|-------|---------|
| `ps aux` | מציג את כל התהליכים הרצים |
| `ps aux \| grep NAME` | מסנן תהליכים לפי שם |
| `top` | תהליכים בזמן אמת (q לצאת) |
| `kill PID` | הורג תהליך לפי מספר |
| `kill -9 PID` | הורג תהליך בכוח |

## רשת (Network)

| פקודה | מה עושה |
|-------|---------|
| `ifconfig` | מציג ממשקי רשת וכתובות IP |
| `ping HOST -c 3` | בודק חיבוריות (3 פעמים) |
| `lsof -i :PORT` | מי מאזין על port מסוים |
| `nslookup DOMAIN` | מחזיר IP של domain |
| `curl ifconfig.me` | מציג IP ציבורי |

## הרשאות (Permissions)

| פקודה | מה עושה |
|-------|---------|
| `ls -la` | מציג קבצים עם הרשאות |
| `chmod 755 FILE` | משנה הרשאות |

### קריאת הרשאות
```
-rw-r--r--
│ │  │  │
│ │  │  └── others (כולם)
│ │  └── group (קבוצה)
│ └── owner (בעלים)
└── סוג (- קובץ, d תיקייה)
```

### ערכי chmod
- r = 4 (read)
- w = 2 (write)
- x = 1 (execute)
- 755 = rwxr-xr-x
- 644 = rw-r--r--

## Logs

| פקודה | מה עושה |
|-------|---------|
| `tail FILE` | 10 שורות אחרונות |
| `tail -f FILE` | עוקב בזמן אמת (Ctrl+C לצאת) |
| `grep "TEXT" FILE` | מחפש טקסט בקובץ |

## Filesystem

| פקודה | מה עושה |
|-------|---------|
| `pwd` | תיקייה נוכחית |
| `df -h` | מקום פנוי בדיסק |
| `du -sh DIR` | גודל תיקייה |

### תיקיות חשובות

| תיקייה | תוכן |
|--------|------|
| `/` | root - שורש המערכת |
| `/home` | תיקיות משתמשים |
| `/etc` | קבצי הגדרות |
| `/var/log` | קבצי log |
| `/tmp` | קבצים זמניים |
| `/usr/bin` | תוכנות מותקנות |
